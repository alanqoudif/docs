</Note>

## Base URL

في الإنتاج استخدم دومين مبيان:

```text
https://mibyanai.com
```

في التطوير المحلي استخدم Vite dev server:

```text
http://localhost:5173
```

## Authentication

معظم المسارات تتطلب Supabase access token:

```http
Authorization: Bearer <supabase_access_token>
```

## أهم المسارات

<CardGroup cols={2}>
<Card title="Agent run" icon="bolt" href="/developers/api-overview">
`POST /api/public/agent-run` يشغل حلقة الوكيل ويبث SSE.
</Card>

<Card title="Classifier" icon="route">
`POST /api/public/classify` يصنف الطلب ويقترح الوكيل والأسئلة.
</Card>

<Card title="Skills" icon="puzzle-piece">
`GET/POST /api/public/skills` يعرض المهارات أو يختار skill مناسباً.
</Card>

<Card title="AI models" icon="brain">
`GET /api/public/ai/models` يعرض النماذج المسموحة لتكاملات AI proxy.
</Card>
</CardGroup>

## OpenAPI

ملف `api-reference/openapi.json` يحتوي مواصفة أولية لأهم المسارات. ليست كل مسارات التطبيق موثقة فيه بعد، لكنها لم تعد مواصفة Plant Store الافتراضية.
