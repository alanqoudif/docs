<Note>
واجهة API العامة لمبيان لا تزال قيد التجهيز. سنضيف هنا طريقة الاستخدام، المصادقة، والأمثلة العملية عندما تصبح الواجهة مستقرة للمستخدمين والمطوّرين.
</Note>

## لمن هذه الصفحة؟

هذه الصفحة للمستخدمين أو الشركاء الذين يريدون ربط مبيان بتطبيقاتهم، أنظمة داخلية، أو أدوات تشغيل خارجية.

## ما الذي سيضاف لاحقاً؟

- طريقة إنشاء مفتاح API أو الحصول على صلاحية الوصول.
- أمثلة لإرسال طلب وإنشاء مخرج.
- أمثلة للقراءة والتصدير.
- حدود الاستخدام وسياسات الأمان.
- رسائل الأخطاء الشائعة وطريقة التعامل معها.

## حالياً

إذا كنت تحتاج إلى تكامل API الآن، تواصل مع فريق مبيان واشرح حالة الاستخدام المطلوبة:

- ما التطبيق أو النظام الذي تريد ربطه؟
- ما نوع المخرجات المطلوبة؟
- هل تحتاج إلى إنشاء مستندات، عروض، جداول، أو مواقع؟
- ما حجم الاستخدام المتوقع؟
