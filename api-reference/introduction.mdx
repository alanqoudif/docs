<Note>
  The public developer API is still being prepared. This reference documents the routes that actually exist in `atlas-swarm-core-1/src/routes/api/public`.
</Note>

## Base URL

In production, use the Mibyan domain:

```text
https://mibyanai.com
```

In local development, use the Vite dev server:

```text
http://localhost:5173
```

## Authentication

Most routes require a Supabase access token:

```http
Authorization: Bearer <supabase_access_token>
```

## Key routes

<CardGroup cols={2}>
<Card title="Agent run" icon="bolt" href="/developers/api-overview">
`POST /api/public/agent-run` runs the agent loop and streams SSE.
</Card>

<Card title="Classifier" icon="route">
`POST /api/public/classify` classifies the request and suggests the agent and questions.
</Card>

<Card title="Skills" icon="puzzle-piece">
`GET/POST /api/public/skills` lists skills or chooses a suitable skill.
</Card>

<Card title="AI models" icon="brain">
`GET /api/public/ai/models` lists allowed models for AI proxy integrations.
</Card>
</CardGroup>

## OpenAPI

The `api-reference/openapi.json` file contains an initial specification for the most important routes. Not every app route is documented there yet, but it is no longer the default Plant Store specification.
