## قاعدة التسمية في الوثائق

نستخدم "مبيان" عند الحديث عن المنتج، و"Atlas Swarm Core One" عند الحديث عن المنصة أو المستودع، و"أطلس" عند الحديث عن الوكيل المنسق. إذا ظهر `master` في API أو قاعدة البيانات أو الكود، فهو الاسم البرمجي لأطلس وليس وكيلاً إضافياً.

## مصطلحات المنتج

- `projectId`: معرف المحادثة أو مساحة العمل الحالية. يظهر في المسار `/chat/$projectId`.
- Artifact: المخرج القابل للعرض أو التعديل، مثل مستند، صفحة HTML، جدول، عرض، أو موقع.
- Agent run: دورة تنفيذ الوكيل عبر `/api/public/agent-run`، وتشمل اختيار الأداة، تشغيلها، وإرسال الأحداث للواجهة.
- Tool call: استدعاء دالة داخل حلقة الوكيل مثل `create_doc` أو `create_website`.
- Skill: تعليمات قابلة للحقن في سياق الدردشة، مخزنة في جدول `skills` ويمكن استيرادها عبر API.
- Edge function: وظيفة Supabase منفصلة عن تطبيق TanStack، مثل `generate-deck` و`export-deck-pptx`.

<Note>

</Note>
