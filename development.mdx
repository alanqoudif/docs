## Run the docs site

The `docs` directory is the external Mintlify site. Run it from the root of this directory:

```bash
npm i -g mint
mint dev
```

## Run the app

The actual application lives in `atlas-swarm-core-1`.

```bash
cd atlas-swarm-core-1
pnpm install
pnpm dev
```

Commands available in `package.json`:

- `pnpm dev`: run the Vite dev server.
- `pnpm build`: build for production.
- `pnpm build:dev`: build in development mode.
- `pnpm preview`: preview the build.
- `pnpm lint`: ESLint.
- `pnpm format`: Prettier.

## Requirements

- A recent Node version compatible with Vite 7.
- pnpm.
- A Supabase project with the required tables and keys.
- An OpenRouter API key for running agents.
- A Lovable API key for some AI, email, and export routes.

## Important files

- `vite.config.ts`: TanStack Start and Cloudflare/Vite setup.
- `wrangler.jsonc`: Cloudflare Worker deployment.
- `supabase/config.toml`: local/project Supabase configuration.
- `supabase/migrations`: database migrations.
- `supabase/functions`: deck/export functions.
- `src/routes/api/public`: API routes.

## Quick check

After configuring environment variables:

```bash
pnpm lint
pnpm build
```

<Note>
Do not commit secrets such as `SUPABASE_SERVICE_ROLE_KEY` or `OPENROUTER_API_KEY`. Always review any `.env` file before committing.
</Note>
