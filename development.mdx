## تشغيل موقع الوثائق

مجلد `docs` هو موقع Mintlify الخارجي. شغّله من جذر هذا المجلد:

```bash
npm i -g mint
mint dev
```

## تشغيل التطبيق

التطبيق الفعلي موجود في `atlas-swarm-core-1`.

```bash
cd atlas-swarm-core-1
pnpm install
pnpm dev
```

الأوامر الموجودة في `package.json`:

- `pnpm dev`: تشغيل Vite dev server.
- `pnpm build`: بناء الإنتاج.
- `pnpm build:dev`: بناء بوضع development.
- `pnpm preview`: معاينة build.
- `pnpm lint`: ESLint.
- `pnpm format`: Prettier.

## المتطلبات

- Node حديث متوافق مع Vite 7.
- pnpm.
- Supabase project مع الجداول والمفاتيح.
- OpenRouter API key لتشغيل الوكلاء.
- Lovable API key لبعض مسارات AI/email/export.

## ملفات مهمة

- `vite.config.ts`: إعداد TanStack Start وCloudflare/Vite.
- `wrangler.jsonc`: نشر Cloudflare Worker.
- `supabase/config.toml`: إعداد Supabase المحلي/المشروع.
- `supabase/migrations`: قاعدة البيانات.
- `supabase/functions`: وظائف deck/export.
- `src/routes/api/public`: API routes.

## تحقق سريع

بعد ضبط المتغيرات:

```bash
pnpm lint
pnpm build
```

<Note>
لا تلتزم أي مفاتيح سرية مثل `SUPABASE_SERVICE_ROLE_KEY` أو `OPENROUTER_API_KEY`. ملف `.env` الموجود يجب مراجعته دائماً قبل أي commit.
</Note>
